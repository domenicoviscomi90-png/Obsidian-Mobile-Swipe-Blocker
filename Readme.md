Mobile Swipe Blocker
Mobile Swipe Blocker is a lightweight, essential plugin for Obsidian designed to refine the navigation experience on mobile devices (Android and iOS).

🚀 The Problem
On Obsidian Mobile, a horizontal swipe-right gesture automatically triggers the left sidebar. While useful, this often happens accidentally while scrolling through long notes or trying to select text, leading to a disrupted workflow.

✨ Key Features
Intelligent Interception: Uses a vector-based algorithm to distinguish between vertical scrolling and intentional horizontal swiping.

Accidental Trigger Prevention: Automatically blocks the sidebar from sliding out if the horizontal movement is detected as the primary gesture.

Context Awareness: The blocker automatically disables itself if the left sidebar is already open, ensuring you can still interact with your file explorer or search results normally.

Native Performance: Built with pure JavaScript using Obsidian's native DOM events to ensure zero lag and high responsiveness.
